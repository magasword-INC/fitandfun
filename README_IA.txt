INSTALLATION DE L'IA (STABLE DIFFUSION) - MODE CPU
==============================================================================

J'ai préparé tout l'environnement pour que vous puissiez générer des avatars IA
directement sur votre serveur, en utilisant le processeur (CPU).

Pour finaliser l'installation (téléchargement des modèles), vous devez lancer
une seule commande dans votre terminal.

1. Ouvrez un terminal (ou connectez-vous en SSH).
2. Lancez la commande suivante :

   /home/pouximixi-projet/ai-tools/launch_cpu_mode.sh

3. Laissez tourner !
   - Le script va télécharger environ 4 à 10 Go de données.
   - Cela peut prendre 10 à 30 minutes.
   - Une fois terminé, vous verrez un message indiquant que l'IA tourne sur :
     http://127.0.0.1:7860

4. Laissez ce terminal ouvert (ou utilisez "screen" pour le laisser en fond).

5. Allez sur votre site, page "Mon Profil", et testez la génération !

NOTE :
- La génération sur CPU prendra entre 1 et 3 minutes par image.
- Si vous fermez le terminal, l'IA s'arrêtera.

Bon amusement avec Fit&Fun AI !
